# Skills for Claude

Claude用スキルのコレクションです。

## スキル一覧

### hr-search
人事・労務に関する行政公式サイト限定検索スキル。
厚生労働省・年金機構・e-Gov・裁判所など35の公式サイトを対象に検索し、正確性が高い回答を表示します。

→ [SKILL.md を見る](skills/hr-search/SKILL.md)

## インストール方法
1. 各スキルの `.skill` ファイルをダウンロード
2. Claude画面左側の[カスタマイズ]をクリック
3. Claude画面左側の[スキル]をクリック
4. Claudeのスキル設定画面の[＋]をクリック
5. [スキルを作成]をクリック
6. [スキルをアップロード]をクリック
7. 手順1の`.skill` ファイルをアップロード
